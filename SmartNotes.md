## SmartNotes

Je suis suis étudiante en BUT informatique. Je connais un peu de java, de php, de javascript, de vue.js.
Je débute en mobile. J'ai commencé un cours de Kotlin + jetpack compose récemment.

Je veux créer une application android pour pratiquer le code en techno mobile et mieux comprendre l'architecture des appli mobiles.

## Utilités de l'application :

- Je dois pouvoir charger une photo de mon portable
- Cette photo, représentant du texte manuscrit ou une capture écran de texte, doit être traitée par l'IA pour la transcrire sous forme de texte, à renvoyer dans l'application 
- Ce texte récupéré doit pouvoir être éditable : modifier le texte, mettre en gras, souligner, surligner, faire des puces
- Le texte pourra être sauvegardé sous forme de note.
- Toutes les notes créées pourront être classées dans des catégories
- Toutes les notes pourront porter un ou plusieurs tags
- Je dois pouvoir créer, modifier et supprimer des catégories et des tags.
- J'aimerais que la sauvegarde de mes notes, catégories et tags se fasse en base de données.
- Sans envoi de photo à traiter, je dois pouvoir créer une note avec du texte.

## Avenir du projet :
- J'aimerais plus tard créer une application web me permettant d'accéder à toutes mes notes

## Limites et cadre :
- Je vais coder sous Android Studio
- J'aimerais utiliser Kotlin + Jetpack compose
- Je ne veux pas de backend trop compliqué mais je veux de la sécurité notamment pour ma clé API IA.
- J'ai envie de coder moi même. Guide moi pour que je fasse les tâches moi même, en étant pédagogue, comme sous la forme d'un TD.
